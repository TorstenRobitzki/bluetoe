Detailed Description

A Security manager that implements only LESC pairing.

This is the current default. The SM will reject every attempt to pair to the device without LE Secure Connections.

By default, this security manager only implements "just works" pairing. To implement other pairing methods, the library must get some details about how to handle IO.
