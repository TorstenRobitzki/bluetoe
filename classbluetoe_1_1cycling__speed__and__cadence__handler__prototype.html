◆ set_cumulative_wheel_revolutions()

void bluetoe::cycling_speed_and_cadence_handler_prototype::set_cumulative_wheel_revolutions ( std::uint32_t  new_value)

Function will be called, when the cumlative wheel revolution have to be changed.

The update of the value have to be confirmed by calling confirm_cumulative_wheel_revolutions(). Calling confirm_cumulative_wheel_revolutions() can be done synchronous, or asynchroiunous. It's save to call confirm_cumulative_wheel_revolutions() from an ISR.
